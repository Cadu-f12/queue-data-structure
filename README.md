# C Queue CLI 🚀

Uma implementação robusta de estrutura de dados de fila (Queue) em C, acompanhada de uma interface interativa via terminal.

## 📋 Sobre o Projeto
Este repositório contém a implementação de uma fila dinâmica/estática, focada em demonstrar conceitos fundamentais de gerenciamento de memória e lógica de estruturas de dados. O projeto foi estruturado para ser modular, separando a lógica da fila da interface de interação com o usuário.



## ✨ Funcionalidades
A CLI permite executar as seguintes operações:
* **Enqueue:** Adicionar um elemento ao final da fila.
* **Dequeue:** Remover o elemento do início da fila.
* **Front:** Visualizar o primeiro elemento sem removê-lo.
* **Display:** Exibir todos os elementos da fila atual.
* **Clear:** Esvaziar a fila completamente.

## 🛠️ Tecnologias Utilizadas
* Linguagem C
* GCC (GNU Compiler Collection)
* Makefile (Opcional)
